# java_mini-projects

Коллекция небольших учебных проектов на **Core Java**. Без Spring, без БД — только синтаксис, алгоритмы, ООП и работа с консолью / файлами.

## Содержание

| Проект | Описание |
|--------|----------|
| `Money_Converter` | Конвертер валют (RUB, USD, EUR).|
| `Task Manager` | Консольный трекер задач: добавление, просмотр, удаление задач. Данные хранятся в файле. |
| `CosmoTasks` | Небольшие задачки для практики языка |
| `LearningTask` |  Небольшие задачки для практики языка |

## Технологии

- **Java 21+**
- Только стандартная библиотека (без внешних зависимостей)
- Сборка через `javac` или любую IDE (IntelliJ IDEA, VS Code)

## Как запустить

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/MeTaLdiCODING/java_mini-projects.git

Перейдите в папку нужного проекта:

bash
cd java_mini-projects/Money_Converter
Скомпилируйте и запустите:

bash
javac Main.java
java Main
Или откройте любой проект в IntelliJ IDEA и запустите main метод.
